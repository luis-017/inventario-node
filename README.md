# 📦 Sistema de Inventario - Node.js + MySQL

## 📝 Descripción
Aplicación web para gestión de inventario que permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de productos.

## 🚀 Tecnologías utilizadas
- **Backend**: Node.js, Express.js
- **Frontend**: HTML5, CSS3, JavaScript
- **Base de datos**: MySQL
- **Control de versiones**: Git y GitHub

## 📋 Requisitos previos
- Node.js (v14 o superior)
- MySQL (v8 o superior)
- Git
